# 8DoorsTestMod

## Installation Guide

1. install BepinEx [here](https://github.com/BepInEx/BepInEx/releases/)
2. open the game and close it to get BepinEx fully installed
3. install and unzip the mod into the BepinEx\Plugins Folder inside your game directory

## Usage Guide

1. press F1 once in game to open the menu
2. select what weapon/ability/or location to teleport to in the menu
3. then select add ability/weapon/teleport
4. you can reset weapons and abilities by clicking the reset weapon/ability buttons
